# JavaScript Arrays + Loop Challenge

## Overview

In this assignment, you will practice combining **array methods and loops** to solve problems.

You will also use:
- **string methods**
- **number methods**
- **conditional statements**
- **functions**

This assignment builds on what we practiced as a class — now it’s your turn to apply those skills **independently**.

All work should be completed in the provided `script.js` file.

## Rules & Expectations

- **Work independently** (no copying)
- **Do not rename any arrays or variables**
- Use the **methods and tools we learned in class**
- Follow directions carefully for each challenge
- Use `console.log()` to test your results

---

## What You Will Practice

By completing this assignment, you will practice:

- Searching arrays using `includes()` and `indexOf()`
- Sorting arrays using `sort()`
- Copying arrays using the spread operator `...`
- Looping through arrays using `for`, `while`, and `forEach()`
- Using conditionals (`if / else`)
- Writing and calling functions
- Using string methods like:
  - `.toUpperCase()`
  - `.toLowerCase()`
- Using number methods like:
  - `Math.round()`

## Important Reminders

- Some methods (like `sort()` and `reverse()`) **change the original array**
- Use the **spread operator (`...`)** when you need to keep the original unchanged
- Loops help you work with **each item one at a time**
- Always think:
  👉 *“What is this question asking me to do with the array?”*
